## [v0.1.1] - 2026-03-21

### 新增功能 (Added)
* 新增 Windows Rust CLI capability-family 命令面：`pipeline`、`query`、`report`、`exchange`。
* 新增 `RuntimeSession` 与 `PipelineClient`、`QueryClient`、`ReportClient`、`TracerExchangeClient` 宿主边界。
* 新增 `handler/session` 级测试基座，覆盖 `pipeline`、`query`、`report`、`exchange` 与 `chart` 家族。

### 技术改进/重构 (Changed/Refactor)
* 重构 Windows Rust CLI 命令模型，将旧顶层 `convert`、`import`、`ingest`、`validate`、`export`、`tree`、`crypto` 收敛为 capability 对齐的 family 命令。
* 重构 `apps/tracer_cli/windows/rust_cli/src/commands/handlers/` 目录结构，从平铺 handlers 改为 `pipeline/*`、`query/*`、`report/*`、`exchange/*` 与 `chart/*`。
* 重构 `apps/tracer_cli/windows/rust_cli/src/core/runtime.rs`，让 `invoke.rs` 仅保留 transport/FFI 执行职责。
* 重构 Windows CLI 活动文档入口，补齐 `docs/time_tracer/clients/windows_cli/README.md` 与 `docs/time_tracer/clients/windows_cli/specs/STRUCTURE.md`。

### 修复 (Fixed)
* 修复 Windows CLI suite、framework 与 quality gate 对旧命令形态的调用，统一切换到 canonical `--db`、`report render`、`report export` 与 `exchange` 命令。
* 修复 `chart` family 缺少独立 session 注入测试边界的问题，补齐 `chart/mod.rs` 目录根形态与最小会话测试口。

### 弃用/删除 (Deprecated/Removed)
* 删除旧兼容入口 `blink`、`zen`、`--database`、`--out`、`--project`、`remark-day` 与 `sensitive`。
* 删除 `apps/tracer_cli/windows/rust_cli/src/cli/crypto.rs`、`apps/tracer_cli/windows/rust_cli/src/cli/export.rs`、`apps/tracer_cli/windows/rust_cli/src/cli/tree.rs` 与 `apps/tracer_cli/windows/rust_cli/src/cli/workflow.rs` 主路径。
* 删除旧平铺 handler 入口，统一改由 family 目录承载主线实现。
