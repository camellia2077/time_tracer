## [v0.1.1] - 2026-03-21

### 技术改进/重构 (Changed/Refactor)
* 重构 `crypto decrypt` 的 `tracer exchange` 语义：命令现在执行事务式完整导入，直接更新 active text root、应用包内 `converter TOML` 并全量重建数据库，不再沿用旧的解包输出目录语义。
* 重构 `crypto inspect` 与 `crypto encrypt` 契约到 `tracer exchange package v3`，统一输出严格的 `payload/<year>/YYYY-MM.txt` canonical 路径布局。
* 调整 `crypto` 子命令输出文案，导入完成反馈改为明确展示 `Updated active text root`、`Replaced months`、`Preserved local months` 与 `Database rebuilt`。

### 修复 (Fixed)
* 修复 Windows CLI `crypto decrypt` 只解包 package、不会把包内 TOML 与 TXT 作为一个事务整体导入的问题。
* 修复 `crypto inspect` 仍按旧包版本断言 `package_version = 2` 的问题，统一切换到 `v3` 契约。

### 弃用/删除 (Deprecated/Removed)
* 删除 `crypto decrypt --apply-config` 参数与对应兼容分支，不再支持旧的“解包后可选覆盖 config”模式。
* 删除 CLI 对旧 `tracer exchange v2` 导入语义的兼容预期，不再把 `decrypt` 当成“导出到自定义目录”的前端接口。
