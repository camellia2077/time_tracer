## [v0.2.0] - 2026-04-08

### 新增功能 (Added)
* 新增 Windows CLI `txt view-day` 命令，支持从月度 TXT 文件中按 `--day MMDD` 提取单日 block 内容。

### 技术改进/重构 (Changed/Refactor)
* 重构 Windows CLI TXT 处理边界，新增 `TxtClient` 并接入 shared TXT runtime family；CLI 现在负责输入文件读取、`selected_month` 推断、参数校验与 host 侧错误文案，day-block 解析语义统一委托给 core/runtime 的 `resolve_day_block` 能力。
