## [v0.9.4] - 2026-03-28

### 技术改进/重构 (Changed/Refactor)
* 重构 `exchange import` 的 active converter config 安装链路，将“写入配置并刷新运行态”收敛到 core capability，避免调用方手动维护缓存失效点。
* 收敛 `libs`、shell/C ABI、Windows CLI 之间的 exchange / validation / config 刷新边界，减少配置覆盖、导入校验、运行态刷新等逻辑在表现层重复实现。
* 统一部分 pipeline / reporting / exchange 相关接口、签名与类型风格，清理一批由静态检查暴露出的职责分散、命名不一致与样式漂移问题。

### 修复 (Fixed)
* 修复导入 `.tracer` 后 converter TOML 已覆盖到磁盘、但运行态未立即按新配置重新校验 TXT 的问题；导入后可直接基于新 alias 继续执行 `validate`、`query` 与后续 ingest。
* 修复 exchange import 回归与当前 TXT 年目录布局不一致的问题，统一按 `input/full/YYYY/YYYY-MM.txt` 校验导入后的文本产物。
