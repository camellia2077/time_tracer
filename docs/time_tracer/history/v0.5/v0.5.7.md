## [v0.5.7] - 2026-02-08

### 新增功能 (Added)
* **Validation**: 新增单活动时长上限校验（>16 小时触发错误），并支持在事件备注中通过 `@allow-long` 放行。
* **Docs**: 新增输入文本规则说明文档 `docs/time_tracer/guides/input/input_format_cn.md`，覆盖格式约束与 `@allow-long` 用法。
* **Query**: 新增 `query data days-duration` 与 `query data days-stats`，支持按日总时长排序与统计（均值/方差/标准差/分位数等）。
* **Docs**: 新增 CLI 命令总览页 `docs/time_tracer/guides/cli/command_index.md` 与各命令独立说明文档。
* **Reports**: 新增 `month`/`recent`/`week`/`year` 报表统计项，展示 `status`/`sleep`/`exercise`/`cardio`/`anaerobic` 为 true 的天数及占比（`md`/`typ`/`tex`）。
* **Docs**: 新增数据库说明 `docs/time_tracer/guides/database/database_schema.md`，覆盖表结构、字段语义、索引与常用查询。

### 技术改进/重构 (Changed/Refactor)
* **Validation**: 引入 `SourceSpan` 与 `Diagnostic`，结构校验与逻辑校验统一输出文件路径与行号，并回显原始行文本。
* **Reports**: `TimeFormatDuration` 统一为 `Xh Ym` 输出，小时/分钟为 0 也显示单位（影响 `md/tex/typ` 报表）。
* **CLI**: 管道输出去除表情与字符画，改为纯文本标识。
* **Docs**: `docs/time_tracer/guides/` 按职责拆分目录（`cli/`、`input/`、`build/`、`native/`、`config/`）。
* **Schema**: 集中管理 JSON/DB 键名，统一 `days`/`time_records`/`projects` 相关 SQL 列名与 `headers` 输出键（含 `cardio`、`anaerobic`）。
