## [v0.3.8.1] - 2025-08-31

### 技术改进/重构 (Changed/Refactor)
* **解析架构重构 (Intermediate JSON)**:
    * **引入 JSON 中间态**: 彻底改变了旧版本直接验证 TXT 格式的解析方式。系统现在先将原始纯文本 (`.txt`) 转换为 JSON 格式，随后基于 JSON 结构进行语义分析、校验与入库。
    * **解析逻辑优化**: 利用 JSON 树状结构的优势极大地简化了解析器实现，显著提升了系统的稳定性与扩展性。
    * **校验前移**: 数据的合法性验证现在在 JSON 转换阶段完成，解析效率更高。
