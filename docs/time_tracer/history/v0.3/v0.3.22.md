## [v0.3.22.7] - [v0.3.22.5]

### 新增功能 (Added)
* **娱乐统计扩展**:
    * 实现 `recreation` 相关字段的数据生成与持久化存储。
    * 为日报告增加了 **Markdown**, **LaTeX**, **Typst** 格式的娱乐时长导出。
    * 对 JSON 配置增加了相应的 `recreation` 规则验证。


## [v0.3.22.4]

### 技术改进/重构 (Changed/Refactor)
* **工具类整合**: 移除了 `query_utils`，将其核心功能合并至 `TimeFormat` 模块。


## [v0.3.22.3]

### 技术改进/重构 (Changed/Refactor)
* **LaTeX 逻辑统一**: 扩展了 **`IFormattingStrategy**` 接口，将 **LaTeX** 的具体渲染实现统一整合进 **`ProjectTreeFormatter**`。


## [v0.3.22.2]

### 技术改进/重构 (Changed/Refactor)
* **数据规范化**: 查询相关的 JSON 配置文件全面改用 **snake_case** 命名法。


## [v0.3.22.1]

### 技术改进/重构 (Changed/Refactor)
* **代码风格标准化**: 按照 **Google C++ Style Guide**，将所有类成员变量命名统一加上下划线后缀 `_`。


## [v0.3.22.0]

### 技术改进/重构 (Changed/Refactor)
* **工厂模式重构**: 引入基于注册机制的工厂模式，彻底消除硬编码的 `switch` 语句，显著提升代码对 **开闭原则 (OCP)** 的符合度。
