## [v0.10.1] - 2026-04-15

### 新增功能 (Added)
* 新增 report breakdown 聚合查询能力，让 `libs/tracer_core` 可以直接按时间窗口返回 root 级别的总时长与占比数据，供上层 `pie` 与 `horizontal bar` 视图直接消费。

### 技术改进/重构 (Changed/Refactor)
* 重构 `libs/tracer_core/src/infra/query/data/` 的 report query 映射与 orchestrator 链路，把 `trend` 的按日序列查询和 `breakdown` 的按 root 聚合查询拆成两条明确的数据语义。
* 收敛 `query_runtime_service_report_mapping.cpp` 与相关 request / dispatch 入口，使 runtime query 可以基于同一套 report 时间窗口规则处理 `day|week|month|year|range|recent` 的 breakdown 聚合。

### 修复 (Fixed)
* 修复 pie 类图表长期复用按日 trend series 导致语义不正确的问题，避免在多日窗口里把“每天变化”误显示成“root 构成”。
* 修复 `day` 与多日窗口在 root 聚合展示上的行为漂移问题：单日可以保留完整 root breakdown，多日窗口继续维持更适合图表展示的聚合口径。
