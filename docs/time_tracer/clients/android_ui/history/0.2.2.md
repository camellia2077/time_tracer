## [v0.2.2] - 2026-03-01

### Changed/Refactor
* 同步 Android 启动图标风格到 `apps/shared/branding/sharp_rounded_white.svg`（圆角白底箭头）。
* 重构 `Config` 信息入口为单一 `About` 入口，整合项目信息与许可证列表。
* 重构 `About` 页面结构：统一展示 Author、Repo、Core、App 与第三方许可证详情。
* 增加 AboutLibraries 自动生成许可证元数据，减少手工维护偏差。
* 同步多语言文案：`values`、`values-zh`、`values-ja` 补齐 `About` 与许可证相关键值。

### Fixed
* 修复 `Open About` 点击闪退：消除可滚动父容器与 `Scaffold + weight` 组合导致的无限高度约束问题。
* 修复许可证元数据不可用时崩溃：改为失败兜底提示“许可证暂时不可用”。

### Added
* 新增 About 页面许可证浏览能力：支持按库查看 License 详情。

### Deprecated/Removed
* 下线分散入口：不再使用多个独立按钮分别展示项目信息与许可证。
