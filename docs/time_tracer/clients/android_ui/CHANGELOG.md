# Android 更新日志 (Android Changelog)

本文档记录 `tracer_android` 移动端应用的所有用户侧版本更新。

---

## [0.2.0] - 2026-02-22
### Added (新增)
* **图表可视化模式**：支持生成折线图、柱状图及饼图，直观展现时间分配趋势。
* **多语言支持**：集成中文、英文、日语三语动态切换。
* **配置导入/导出**：支持 TOML 配置文件及原始数据文件的备份与迁移。
* **Material Design 3 适配**：全面引入 M3 设计语言，提供现代化的 UI 交互体验。

---

## [0.1.0] - 2026-02-14
### Added (新增)
* **核心功能建立**：支持通过移动端进行活动录入、日志解析及报告生成。
* **JNI 桥接**：实现与 C++ 核心引擎的高效数据交换。
