# Windows CLI 更新日志 (Windows CLI Changelog)

本文档记录 `tracer_windows_cli` 客户端的用户侧版本更新。

---

## [0.2.0] - 2026-02-24
### Added (新增)
* **加密/解密进度条支持**：执行 `crypto encrypt` 或 `crypto decrypt` 时，支持实时进度显示，长任务过程更可见。
* **双层进度展示**：同时显示总体进度与当前目录（如年份目录）进度，便于判断全局与局部处理状态。
* **进度信息增强**：支持显示已处理文件数、处理速度、剩余数据量和预计剩余时间（ETA）。
