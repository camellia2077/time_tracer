## [v0.1.0] - 2026-02-24

### Added
* 发布 Rust 主线 Windows CLI 首个稳定版本，命令入口覆盖 `ingest/query/export/tree/doctor/licenses/crypto/chart`。
* 新增 `chart` 多图表类型与 HTML 导出能力。
* 新增 `crypto` 处理进度反馈能力。

### Changed/Refactor
* 重构 Windows CLI 实现到 Rust 2024。
* 重构参数解析到 `clap` 声明式模型。
* 重构 Core 调用链路：通过 `libloading` 动态加载 `tracer_core.dll` 并统一走 C ABI。
* 重构命令结构：参数定义集中在 `src/cli/mod.rs`，处理逻辑集中在 `src/commands/handlers/*.rs`。
* 重构错误模型：统一到 `src/error/mod.rs`，便于自动化测试与结果解析。

### Deprecated/Removed
* 下线历史手写 Windows CLI 实现，主线仅保留 Rust CLI。
