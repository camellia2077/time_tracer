# Time Tracer 品牌色彩设计理念 (Color Philosophy)

本文档阐述了 Time Tracer 品牌主色调的选取逻辑及其在动态图形与残影中的物理衰减法则。我们的色彩设计摒弃了纯粹依赖肉眼感觉的随意调色，而是一门关于能量守恒与物理耗散的严谨科学。

## 1. 核心双色渐变 (The Core Gradient)

Time Tracer 的基础品牌色由两种极高能量的色彩构成，它们通过紧密的线性渐变（Linear Gradient）构建了图标的主体（即 100% 的能量源）。这两者的结合，恰如其分地传达了 **“用绝对精密的物理法则，去展现极速流逝的时间”** 这一核心精神。

*   **起点色：超空间靛蓝 (Hyper-Space Indigo) — `#4F46E5`**
    *   **物理寓意**：深邃、精密、绝对的物理秩序。这种偏冷的深蓝色代表着基态的宇宙真空以及时间作为第一维度的不可撼动性。它不带有任何主观情绪，而是冰冷且极具理性的“度量准绳”。
*   **终点色：高频紫罗兰 (High-Frequency Violet) — `#7C3AED`**
    *   **物理寓意**：高能、极速、突破物理限制。在可见光谱中，紫色/紫罗兰色处于最高频率、最高能量的一端。它象征着强大的捕获能力、对时间单位的极致切割，以及全速跃迁向前的爆炸性动能。

## 2. 动态残影与光影衰减 (Energy Decay in Motion)

在表现时间碎片与运动残影时（例如核心箭头图标的尾迹），色彩与不透明度的游走不再是单纯的视觉透视，而是**严格遵循真实世界物理定律的耗散模型**。我们将空间与尺寸的结构规则交给了“黄金分割”，而在此基础之上，我们设计了两种互为表里的光影物理表现哲学：

### 哲学 A：平方反比能量衰减 (Inverse Square Energy Decay)
*   **物理基石**：在三维宇宙中，点光源辐射的能量强度与距离的平方成反比 ($I \propto 1/d^2$)。
*   **视觉应用**：
    *   **绝对当下（100% 主箭头）**：不透明度 $1.0$，凝聚了当前时间的全部质量。
    *   **空间剥离（第一段拖影）**：脱离主体的瞬间，能量发生断崖式下跌，不透明度迅速暴跌至物理残余量（如 `~44%`）。
    *   **彻底耗散（末端残影）**：动能近乎枯竭，以极微弱的游丝附着于空间（不透明度断崖至 `~16%` 甚至更低）。
*   **设计隐喻**：配合黄金序列带来的“空间压缩感”，这种极度凶悍的断崖式非线性光影衰减，完美复刻了处于不可思议的速度下的**高能真理残影**，极具冲击力。

### 哲学 B：光谱偏转与物理冷却 (Spectral Shift & Cooling)
*   **物理基石**：结合多普勒效应 (Doppler Effect) 与热辐射冷却原理。高能发光体在抛弃能量碎片时，其残影不仅仅会变暗，颜色频率更会向红端发生断崖式偏移，并伴随最终冷却后的饱和度丧失（化为灰烬）。
*   **视觉应用**：
    *   **高频·基态 (High Energy State)**：绝对明亮耀眼的蓝紫渐变，是热量与动能最充盈的锋芒。
    *   **中频·红移 (Redshift)**：残影刚刚被抛出，仍带有一丝极速的温度，色彩在光谱上被直接拉扯向红光端（紫红/洋红色），并且伴随大幅度的透明度稀释。
    *   **低频·余烬 (Ash & Ember)**：碎片即将耗散凝固。不仅颜色进一步走向肉眼可见的低频（暗红），其鲜活的色彩也被大量削弱，被融入象征彻底冷却死寂的“暗灰色”。这是时间的抛弃物在虚无中最后的冷火余晖。

---

**总结** 
在 Time Tracer 中：
*   **空间归于几何 (Geometric Space)**：尺寸结构被严密的 1.618 黄金秩序统治。
*   **光影归于物理 (Physical Light)**：色彩的流转与衰退则是反映动能耗散的物理定律。
两者的交响，为这个原本二维的数字图形，注入了令人信服的降维打击般的生命力与绝对速度感。
