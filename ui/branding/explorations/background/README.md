# Background Explorations

本目录用于存放 Time Tracer 图标背景层的探索稿。

## 当前这批方案的目的

- 解决“纯白背景看起来过空”的问题
- 不引入可读文字或高频纹理
- 保持箭头仍然是主识别元素
- 用很浅的品牌色背景提升画面密度
